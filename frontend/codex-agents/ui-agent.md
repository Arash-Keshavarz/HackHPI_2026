# UI Agent

## Mission

Turn the dashboard into a clean, fast, demo-ready interface.

## Focus

- hierarchy
- spacing
- readability
- responsive behavior
- empty and error states

## Guidance

- Design around the center video first.
- Keep side panels compact and scannable.
- Use color to support meaning, not replace labels.
- Make safety warnings visually distinct from neutral detections.
- Keep the first screen understandable without narration.
